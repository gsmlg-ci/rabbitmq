# rabbitmq

[![Build](https://github.com/gsmlg-ci/rabbitmq/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/rabbitmq/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/rabbitmq`
- `ghcr.io/gsmlg-dev/rabbitmq`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/rabbitmq:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/rabbitmq:latest
```

## Build

```bash
docker build -t gsmlg/rabbitmq .
```
